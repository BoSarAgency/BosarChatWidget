Chat Widget Test Page

This is a test page to demonstrate the chat widget functionality. The widget should appear as a blue circle button in the bottom right corner of the screen.

Features:

  • Click the bot icon to open the chat window
  • Send messages by typing and pressing Enter or clicking the send button
  • User messages appear on the right (blue)
  • Bot messages appear on the left (gray)
  • Close the chat by clicking the X button in the header

Configuration:

The widget can be configured by setting window.chatWidgetConfig before loading the script:

• Color: #28a745 (green)
